7. Дифференциальные уравнения

7.1 Дифференциальные уравнения. Основные определения

Дифференциальное уравнение (ДУ) — уравнение, связывающее значение некоторой неизвестной функции в некоторой точке и значение её производных различных порядков в той же точке. Дифференциальное уравнение содержит в своей записи неизвестную функцию, ее производные и независимые переменные; однако не любое уравнение, содержащее производные неизвестной функции, является дифференциальным уравнением.

Обыкновенным дифференциальным уравнением первого порядка называется уравнение вида F(x,y,y')=0, где F — известная функция трех переменных, x — независимая переменная из интервала (a,b), y(x) — неизвестная функция, y'(x) — ее производная. Обыкновенные дифференциальные уравнения, разрешенные относительно производной, т.е. уравнения вида y'=f(x,y) называют уравнениями в нормальной форме.

Функция y=y(x) называется решением дифференциального уравнения, если она непрерывно дифференцируема на (a,b) и при всех x из (a,b) удовлетворяет уравнению F(x,y(x),y'(x))=0.

График решения дифференциального уравнения называют интегральной кривой дифференциального уравнения.

Если дифференциальное уравнение первого порядка y'=f(x,y), имеет решение, то решений у него, вообще говоря, бесконечно много и эти решения могут быть записаны в виде y=y(x,C), где C — произвольная константа. Выражение y(x,C) называют общим решением дифференциального уравнения 1-го порядка: при всех допустимых значениях C функция y=y(x,C) является решением уравнения, y'(x,C) = f(x, y(x,C)); для любого наперед заданного решения y=f(x) найдется такое значение константы C, C=С₀, что y(x,C₀)=f(x).

Однако, если поставить задачу: найти решение, удовлетворяющее условию y(x0)=y0, то при определенных условиях такая задача имеет единственное решение. Задача об отыскании решения y=y(x) дифференциального уравнения y'=f(x,y), удовлетворяющего начальному условию y(x0)=y0, называется задачей Коши. Решение задачи Коши называют частным решением.

7.2 Дифференциальные уравнения с разделяющимися переменными

Дифференциальное уравнение вида y=f(x)·g(y) или M(x)·N(y)+P(x)·Q(y)=0 называется дифференциальным уравнением с разделяющимися переменными. Заметим, что в данных дифференциальных уравнениях каждая из функций зависит только от одной переменной, т.е. происходит разделение переменных.

Для решения такого дифференциального уравнения необходимо домножить или разделить обе части дифференциального уравнения на такое выражение, чтобы в одну часть уравнения входили только функции от x и dx , в другую часть уравнения - только функции от y , dy. Затем в полученном дифференциальном уравнении надо проинтегрировать обе части:

∫ M(x)/P(x) dx + ∫ Q(y)/N(y) dy = 0

Следует заметить, что при делении обеих частей дифференциального уравнения на выражение, содержащее неизвестные x и y, могут быть потеряны решения, обращающие это выражение в ноль. Обратим внимание, что дифференциальные уравнения с разделяющимися переменными легко сводятся к интегрированию. В общем случае получаем два неопределенных интеграла.

Рассмотрим ПРИМЕР решения ДУ с разделяющимися переменными:

y' = 2xy

dy/dx = 2xy

dy/y = 2x·dx

проинтегрируем левую и правую части ДУ, получим:

ln(y) = x2 + C₀

Ответ: y = C·ex²

Всё! Уравнение решено! Отметим, что eC₀ = C — это произвольная постоянная, такая нехитрая замена С₀ на C делается только для того, чтобы укоротить запись решения ДУ.

7.3 Однородное дифференциальное уравнение первого порядка

Дифференциальное уравнение y'=f(x,y) называется однородным относительно x и y, если функция f(x,y) является однородной порядка n, т.е.

f(λx, λy) = λn·f(x, y), λ∈ℝ

Однородную функцию можно представить как функцию от y/x c помощью подстановки: y/x = u(x). Так как y=ux и y'=u'·x+u, то ДУ y'=f(x,y) сводится к уравнению с разделяющимися переменными u и x, которое благополучно решается (не забудьте сделать в конце обратную замену).

Рассмотрим ПРИМЕР решения однородного ДУ:

(2x+y) = x·y'

(2x+y) dx — x dy = 0

Нетрудно заметить, что многочлены при dx и dy, являются однородными функциями первого порядка. Поэтому, данное дифференциальное уравнение является однородным.

Положим y= u(x)·x, тогда y' = u'·x + u и dy = x du + u dx.

ДУ преобразуется к виду: 2x dx — x2 du = 0.

Разделим обе части уравнения на x, получим: x du = 2 dx; du = 2/x dx

Выполняя деление на x, мы могли потерять решение x=0. Прямая подстановка показывает, что x=0 действительно является одним из решений нашего уравнения. Интегрируем последнее выражение. Получим: u = 2 ln|x| + C. Вернемся к старой переменной и запишем ответ.

Ответ: y = x·(2·ln|x|+C), x = 0 — два решения уравнения.

7.4 Линейные дифференциальные уравнения первого порядка

Линейное однородное ДУ первого порядка. Дифференциальное уравнение вида y'+a(x)·y=0 называется линейным однородным ДУ. Данное ЛОДУ может быть решено, как ДУ с разделяющимися переменными, пример решения которого рассмотрен в пункте 7.2. Но чаще общее решение ЛОДУ проще искать в виде:

yoo = C·e−∫ a(x) dx, где yoo означает общее решение однородного линейного ДУ.

Линейное неоднородное ДУ первого порядка y'+a(x)·y=f(x) решается методом вариации произвольной постоянной, который состоит из четырех этапов:

1) сначала находится общее решение соответствующего ЛОДУ в виде: yoo = C·e−∫ a(x) dx;

2) далее ищут частное решеие ДУ, для этого варьируют произвольную постоянную С, то есть, заменяют C функцией С(x) в формуле выше;

3) в заключении функция (частное решение ДУ) yчн = C(x)·e−∫ a(x) dx подставляется в исходное неоднородное дифференциальное уравнение, откуда определяется C(x) и далее записывается выражение для частного решения исходного неоднородного ДУ — yчн.

4) записывается решение исходного ДУ в форме: y=yoo+yчн

Рассмотрим ПРИМЕР решения ЛНДУ 1-го порядка:

y'—x·y = ex²/2 — ЛНДУ первого порядка.

Найдем общее решение соответствующего ЛОДУ y'—x·y = 0: yoo = C·ex²/2

Найдем частное решение исходного ЛНДУ: yчн = C(x)·ex²/2

y'чн = C'(x)·ex²/2 + x·C(x)·ex²/2

Подставим yчн и y'чн в исходное ЛНДУ вместо y и y', соответственно:

C'(x)·ex²/2 = ex²/2

C'(x) = 1; C(x) = x

Тогда частное решение исходного ЛНДУ запишется в виде: yчн = x·ex²/2, т.к. решение уравнения записывается в виде: y=yoo+yчн, запишем решение нашего ЛНДУ первого порядка.

Ответ: y = (C+x)·ex²/2

7.11 ДУ в полных дифференциалах

ДУ вида: P(x,y)dx + Q(x,y)dy= 0 называется дифференциальным уравнением в полных дифференциалах, если существует такая функция двух переменных U(x,y) с непрерывными частными производными, что справедливо выражение: dU(x,y) = P(x,y)dx + Q(x,y)dy

Необходимое и достаточное условие: пусть функции P(x,y) и Q(x,y) имеют непрерывные частные производные в некоторой области D. Дифференциальное уравнение P(x,y)dx + Q(x,y)dy = 0 будет являться ДУ в полных дифференциалах тогда и только тогда, когда справедливо равенство: dQ/dx = dP/dy

Алгоритм решения ДУ в полных дифференциалах

1) Сначала необходимо убедиться, что выполняется необходимое и достаточное условие того, что ДУ является ДУ в полных дифференциалах: dQ/dx=dP/dy

2) Затем запишем систему двух ДУ, определяющих частные производные функции U(x,y) по x и по y :

dU/dx = P(x,y)

dU/dy= Q(x,y)

3) Интегрируем первое уравнение по переменной x. Вместо постоянной C запишем неизвестную функцию, зависящую от y

U(x,y) = ʃ P(x,y)dx + φ(y)

4) Подставляем функцию U(x,y) во второе уравнение и дифференцируем всё это по y:

dU/dy= d(ʃP(x,y)dx+φ(y))/dy = Q(x,y)

Получим отсюда выражение для производной неизвестной функции φ(y) :

φ'(y) = Q(x,y) — d/dy( ʃ P(x,y) dx)

5) Интегрируя последнее выражение находим φ(y) и, следовательно, функцию U(x,y) :

U(x,y) = ʃ P(x,y) dx + φ(y)

6) Общее решение ДУ в полных дифференциалах записывается в виде:

U(x,y) = C

P.S. в пункте 3) вместо интегрирования первого уравнения по переменной x можно интегрировать второе уравнение по переменной y. В этом случае, после интегрирования нужно определить неизвестную функцию φ(x).


ВАЖНЫЙ ПРИМЕР, решение ДУ в полных дифференциалах:

x·y' = (x2/ey) + 2

Преобразуем ДУ к виду:

(—x2 — 2·ey) dx + (x·ey) dy = 0

В данном уравнении:

P(x,y) = —x2 — 2·ey и Q(x,y) = x·ey

Проверка необходимого и достаточного условия:

dP/dy= —2·ey; dQ/dx = ey

Таким образом: dP/dy ≠ dQ/dx условие НЕ выполняется!

Найдем множитель μ(x) такой, что выполняется равенство:

d(μ(x)·P(x,y))/dy = d(μ(x)·Q(x,y))/dx

т.е. такой множитель μ(x), при котором будет выполняться необходимое и достаточное условие:

d(μ(x)·P(x,y))/dy = — 2·μ(x)·ey

d(μ(x)·Q(x,y))/dx = μ'(x)·x·ey + μ(x)·ey

тогда: —2·μ(x)·ey = μ'(x)·x·ey + μ(x)·ey

Т.о. мы получили ДУ для функции μ(x): μ'(x)·x + 3·μ(x) = 0

Решая это ДУ, как ДУ с разделяющимися переменными, получим: μ(x) = 1/(x3)

Домножим на μ(x) исходное дифф. уравнение:

(—1/x — 2·ey·1/x3) dx + (ey·1/x2) dy = 0

ДУ уже является дифференциальным уравнением в полных дифференциалах, т.к. выполняется необходимое и достаточное условие:

dP/dy = dQ/dx = —(2/x3)·ey

Составим систему ДУ:

dU/dx = P(x,y) = —1/x — 2·ey·1/x3

dU/dy= Q(x,y) = 1/x2·ey

Интегрируем первое уравнение системы по x, получим:

U(x,y) = ʃ P(x,y) dx + φ(y)

U(x,y) = ey·1/x2 — ln(x) + φ(y)

Продифференцируем полученное выражение для U(x,y) по переменной y и подставим полученное выражение во второе уравнение системы, получим:

(1/x2)·ey + φ'(y) = (1/x2)·ey

Т.о. φ'(y) = 0, тогда:

U(x,y) = (1/x2)·ey — ln(x)

Общее решение ДУ в полных дифференциалах записывается в виде U(x,y)=C, получим:

(1/x2)·ey — ln(x) = C

ОТВЕТ: y = ln( x2·( ln(x)+C ) )
