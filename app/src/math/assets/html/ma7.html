8. Числовые ряды


8.1 Числовые ряды. Основные определения

Пусть задана бесконечная последовательность чисел U1, U2 ... Un, ... Выражение U1 + U2 +...+ Un + ... называется числовым рядом.

Числа U1, U2 ... Un, ... называются членами этого ряда. Член Un ряда, стоящий на n-ом месте, считая от начала, называется общим членом этого ряда.

Сумма конечного числа n первых членов ряда называется n-ой частичной суммой ряда.

ТЕОРЕМА: если сходится ряд, получившийся из данного ряда отбрасыванием нескольких его членов, то сходится и сам данный ряд. Обратно, если сходится данный ряд, то сходится и ряд, получившийся из данного отбрасыванием нескольких членов.

ТЕОРЕМА: если ряд U1 + U2 + ... + Un + ... сходится и его сумма равна S, то ряд c·U1 + c·U2 + ... + c·Un + ... (где c – произвольное действительное число) также сходится и его сумма равнa с·S.

ТЕОРЕМА: пусть ряд a1 + a2 + ... + an + ... и ряд b1 + b2 + ... + bn + ... сходятся и их суммы, соответственно равны S1 и S2, то ряд (a1+b1) + ... + (an+bn) + ... и ряд (a1-b1) + ... + (an-bn) + ... также сходятся и их суммы равны соответственно S1+S2 и S1-S2.

ТЕОРЕМА (Необходимый признак сходимости ряда): если ряд сходится, то его общий член стремится к нулю при n➝∞.

Заметьте, что это лишь необходимый признак, поэтому если предел общего члена равен нулю ряд может как сходиться, так и расходиться, а вот если предел нулю не равен, то ряд однозначно расходится.

8.3 Сравнение и сходимость знакопостоянных рядов

Пусть даны два ряда с положительными членами U1 + U2 + ... + Un + ... — (1)-й ряд и V1 + V2 + ... + Vn + ... — (2)-й ряд. Для них справедливы следующие утверждения:

ТЕОРЕМА (Первый признак сравнения числовых рядов): пусть члены ряда (1) меньше соответствующих членов ряда (2), т.е. Un ≤ Vn при n=1, 2, ... Тогда, если ряд (2) сходится, то сходится и ряд (1). Иными словами: из сходимости ряда с большими членами следует сходимость ряда с меньшими членами.

ТЕОРЕМА (Второй признак сравнения числовых рядов): пусть члены ряда (1) больше соответствующих членов ряда (2), т.е. Un ≥ Vn при n=1, 2, ... Тогда, если ряд (2) расходится, то расходится и ряд (1). Иными словами: Из расходимости ряда с меньшими членами следует расходимость ряда с большими членами.

Ряд называется знакопеременным, если среди его членов имеются как положительные так и отрицательные. Знакочередующиеся ряды являются частным случаем знакопеременных рядов.

Знакопеременный ряд называется абсолютно сходящимся, если сходится ряд, составленный из абсолютных величин его членов. Если знакопеременный ряд сходится, а ряд, составленный из абсолютных величин его членов, расходится, то данный знакопеременный ряд называется условно сходящимся.
