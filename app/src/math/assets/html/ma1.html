2. Предел функции, непрерывность

2.1 Определение предела по Коши и по Гейне

Определение предела по Коши. Число A называется пределом функции f(x) в точке x=a, если эта функция определена в некоторой окрестности точки a за исключением, быть может, самой точки a, и для каждого ε>0 существует δ>0 такое, что для всех x, удовлетворяющих условию |x–a|<δ, x≠a, выполняется неравенство |f(x)–A|<ε.

Определение предела по Гейне. Число A называется пределом функции f(x) в точке x=a, если для любой последовательности точек xk (k=1..∞), сходящейся к а, но не содержащей а в качестве одного из своих элементов (иными словами в проколотой окрестности точки а), последовательность значений функции f(x) сходится к А.

Если функция f(x) имеет предел в точке a, то этот предел единственный.

2.2 Односторонние пределы

Число A называется пределом функции f(x) СЛЕВА в точке a, если для каждого ε>0 существует δ>0 такое, что для всех x∈(a-δ,a) выполняется неравенство |f(x)-A|<ε

Число A называется пределом функции f(x) СПРАВА в точке a, если для каждого ε>0 существует δ>0 такое, что для всех x∈(a,a+δ) выполняется неравенство |f(x)-A|<ε

Для обозначения предела слева вместо x➝a пишут x➝a- или x➝a-0.

Для обозначения предела справа вместо x➝a пишут x➝a+ или x➝a+0.

Эти пределы характеризуют поведение функции слева и справа от точки a. Их называют односторонними пределами.

2.3 Бесконечные пределы. Бесконечно малые, бесконечно большие функции и их связь

Пусть функция f(x) определена в некоторой проколотой окрестности точки x₀ из области определения этой функции. Будем говорить, что предел функции f(x) в точке a равен ∞, если для любого сколь угодно большого M>0 существует такая δ-окрестность точки a, что для всех x, удовлетворяющих условию |x–a|<δ, x≠a, выполняется неравенство |f(x)|>M, то говорят, что функция f(x) имеет бесконечный предел в точке a и пишут: limx➝a f(x) = ∞.

Если предел фукции f(x) при x➝a равен ∞, то функция f(x) называется бесконечно большой функцией при x➝a.

Если предел фукции f(x) при x➝a равен 0, то функция f(x) называется бесконечно малой функцией при x➝a.

ТЕОРЕМА о связи бескон. большой и бескон. малой функции: пусть f(x) - бесконечно малая функция при x➝a и f(x)≠0 в некоторой проколотой окрестности точки a, тогда g(x)=1/f(x) - бесконечно большая функция при x➝a. И наоборот.

ТЕОРЕМА о произведении бесконечно малой функции на ограниченную: пусть f(x) - бескон. мал. функция при x➝a, а g(x) - ограниченная функция в некоторой проколотой окрестности точки a, тогда f(x)·g(x) - бескон. мал. функция при x➝a.

Свойства бесконечно больших функций:

1) произведение бескон. большой функции при x➝a на постоянную c≠0 есть бескон. большая функция при x➝a;

2) произведение двух бескон. больших функций при x➝a есть бескон. большая функция при x➝a;

3) произведение бескон. большой функции при x➝a на функцию, имеющую конечый предел в точке a, есть бескон. большая функция при x➝a.

2.4 Арифметические свойства пределов функций

ТЕОРЕМА об арифметических свойствах конечных пределов функций: пусть функции f(x) и g(x) определены в некоторой проколотой окрестности точки x₀ из пересечения областей определения этих функций. Пусть также существуют пределы этих фукций при x➝x₀: limx➝x₀ f(x) = A, limx➝x₀ g(x) = B. Тогда существуют:

limx➝x₀ (f(x) ± g(x)) = ( limx➝x₀ f(x) ) ± ( lim x➝x₀ g(x) )

limx➝x₀ (f(x) · g(x)) = ( limx➝x₀ f(x) ) · ( lim x➝x₀ g(x) )

limx➝x₀ (f(x) / g(x)) = ( limx➝x₀ f(x) ) / ( lim x➝x₀ g(x) ), g(x)≠0, B≠0

2.5 Свойства функций имеющих конечные пределы

ТЕОРЕМА об ограниченности функции, имеющей конечный предел: пусть f(x) имеет конечный предел в точке x₀, тогда она ограничена в некоторой проколотой окрестности точки x₀.

ТЕОРЕМА о стабилизации знака: пусть f(x) имеет конечный предел A≠0 в точке x₀, тогда существует некоторая проколотая окрестность точки x₀ такая, что в ней f(x) имеет то же значение, что и A.

ТЕОРЕМА о предельном переходе в неравенствах: пусть существуют конечные пределы функций f(x) и g(x) в точке x₀. Пусть в некоторой проколотой окрестности точки x₀ f(x)≤g(x), тогда

limx➝x₀ f(x) ≤ limx➝x₀ g(x)

ТЕОРЕМА о сжатой переменной: пусть f(x), g(x), h(x) такие, что f(x)≤h(x)≤g(x) в некоторой проколотой окрестности точки x₀. Пусть функции f(x) и g(x) имеют конечные, равные между собой пределы в точке x₀ т.е.

limx➝x₀ f(x) = limx➝x₀ g(x) = A

тогда существует предел функции h(x) в точке x₀ такой, что

limx➝x₀ h(x) = A

ТЕОРЕМА ВЕЙЕРШТРАССА о существовании конечного предела функции в точке: пусть f(x) монотонна и ограничена на промежутке [a,b], тогда в каждой точке x∈(a,b) существуют конечные правосторонние и левосторонние пределы, а в точке x=a существует только правосторонний конечный предел и в точке x=b только левосторонний конечный предел.

2.6 Замечательные пределы

Первый замечательный предел: limx➝0 (sin(x) / x) = 1.

Второй замечательный предел: limx➝∞ (1+1/x)ˣ = e.

Третий замечательный предел: limx➝0 (ln(1+x) / x) = 1.

Четвертый замечательный предел: limx➝0 ((eˣ − 1) / x) = 1.

Пятый замечательный предел (a∈R): limx➝0 ( ((1+x)ᵃ − 1) / x ) = a.

Другие частовстречаемые пределы:

limx➝0 (1 + k / x)mx = emk,

limx➝0 ((aˣ−1) / x) = ln(a),

limx➝0 ((1 / x) · logₐ(x+1)) = 1 / ln(a),

при a>0, a≠1.

2.7 Раскрытие неопределенностей

Пусть для функций f(x) и g(x) пределы при x➝x₀ равны нулю, где x₀ - конечное число, либо бесконечность. Задача нахождения предела отношения функций f(x) и g(x) (т.е. f(x)/g(x)) при x➝x₀ или доказательство его несуществования называется задачей раскрытия неопределенности 0/0.

2.8 Эквивалентные функции

Пусть функции f(x) и g(x) определены в некоторой проколотой окрестности точки x₀, пусть g(x)≠0, тогда если предел limx➝x₀ (f(x) / g(x))

1) равен 0, то говорят, что функция f(x) мала по сравнению с g(x) при x➝x₀;

2) равен k≠0, то говорят, что функции f(x) и g(x) - функции одного порядка при x➝x₀;

3) равен 1, то говорят, что функция f(x) эквивалентна функции g(x) при x➝x₀, пишут: f(x)~g(x).


Примеры эквивалентных функций при x➝0

sin(x) ~ x

tg(x) ~ x

arcsin(x) ~ x

arctg(x) ~ x

eˣ – 1 ~ x

ln(1+x) ~ x

(1+x)ᵃ – 1 ~ a·x


Примеры эквивалентных функций при x➝∞

anxⁿ + an-1xⁿ⁻¹ + … + a₀ ~ anxⁿ

ᵐ√ ( anxⁿ + an-1xⁿ⁻¹ + … + a₀ )  ~  ᵐ√ ( anxⁿ )

ТЕОРЕМА о замене на эквивалентные при вычислении предела отношения: пусть функция f(x)~f₁(x) при x➝x₀ и функция g(x)~g₁(x) при x➝x₀, тогда:

1) Если существует предел отношения функций f₁(x) и g₁(x) при x➝x₀, то существует предел отношения функций f(x) и g(x) при x➝x₀ и limx➝x₀ (f₁(x) / g₁(x)) = limx➝x₀ (f(x) / g(x))

2) и при x➝x₀ f₁(x) / g₁(x) ~ f(x) / g(x)

ТЕОРЕМА о замене на эквивалентные при вычислении предела суммы:

1) Пусть функция f(x) мала по сравнению с функцией g(x) при x➝x₀, тогда f(x)+g(x)~g(x) при x➝x₀.

2) Пусть f(x)~c₁·h(x) и g(x)~c₂·h(x) при x➝x₀, тогда f(x)+g(x)~(c₁+c₂)·h(x) при x➝x₀, если c₁+c₂≠0.

2.9 Правило Лопиталя

Правило Бернулли-Лопиталя — метод нахождения пределов функций, раскрывающий неопределённости вида 0/0 и ∞/∞ . Обосновывающая метод теорема утверждает, что при некоторых условиях предел отношения функций равен пределу отношения их производных.

Правило Лопиталя: если

1) пределы функций f(x) и g(x) оба равны 0 или оба равны ∞ при x➝a

2) функции f(x) и g(x) дифференцируемы в проколотой окрестности точки a

3) g'(x)≠0 в проколотой окрестности точки a

4) и существует предел: limx➝a ( f ' (x) / g ' (x) ), тогда существует limx➝a ( f (x) / g (x) ) = limx➝a ( f ' (x) / g ' (x) )

2.10 Непрерывность функций

Первое определение непрерывности: пусть функция f(x) определена в некоторой окрестности точки x₀. Функция f(x) называется непрерывной в точке x₀, если limx➝x₀ f(x) = f(x₀).

Второе определение непрерывности: пусть функция f(x) определена в некоторой окрестности точки x₀ из области определения функции. Функция f(x) называется непрерывной в точке x₀, если бесконечно малому приращению аргумента соответствует бесконечно малое приращение функции в этой точке.

ТЕОРЕМА об арифметических свойствах непрерывных функций: пусть функции f(x) и g(x) непрерывны в точке x₀, тогда в этой точке непрерывны: сумма f(x)±g(x), произведение f(x)·g(x) и частное f(x)/g(x) этих функций.

ТЕОРЕМА о непрерывности сложной функции: пусть f(y) непрерывна в точке y₀, пусть y=g(x) непрерывна в точке x₀ и g(x₀)=y₀, тогда сложная функция f(g(x)) будет непрерывна в точке x₀.

2.11 Классификация точек разрыва функции

Будем называть точкой разрыва точку x₀ если функция не непрерывна в ней.

Пусть x₀ - точка разрыва функции f(x). Точка x₀ называется:

1) точкой разрыва первого рода, если существуют конечные пределы limx➝x₀− f(x), limx➝x₀+ f(x). При этом величина limx➝x₀+ f(x) − limx➝x₀− f(x) называется скачком функции в точке x₀;

2) точкой устранимого разрыва, если эти пределы конечны и равны;

3) точкой разрыва второго рода, если хотя бы один из этих пределов не существует или равен бесконечности.
