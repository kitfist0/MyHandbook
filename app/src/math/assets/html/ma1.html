2. Предел функции, непрерывность


2.1 Определение предела по Коши и по Гейне

Определение предела по Коши. Число A называется пределом функции f(x) в точке x=a, если эта функция определена в некоторой окрестности точки a за исключением, быть может, самой точки a, и для каждого ε>0 существует δ>0 такое, что для всех x, удовлетворяющих условию |x–a|<δ, x≠a, выполняется неравенство |f(x)–A|<ε.

Определение предела по Гейне. Число A называется пределом функции f(x) в точке x=a, если для любой последовательности точек xk (k=1..∞), сходящейся к а, но не содержащей а в качестве одного из своих элементов (иными словами в проколотой окрестности точки а), последовательность значений функции f(x) сходится к А.

Если функция f(x) имеет предел в точке a, то этот предел единственный.


2.2 Односторонние пределы

Число A называется пределом функции f(x) СЛЕВА в точке a, если для каждого ε>0 существует δ>0 такое, что для всех x∈(a-δ,a) выполняется неравенство |f(x)-A|<ε

Число A называется пределом функции f(x) СПРАВА в точке a, если для каждого ε>0 существует δ>0 такое, что для всех x∈(a,a+δ) выполняется неравенство |f(x)-A|<ε

Для обозначения предела слева вместо x➝a пишут x➝a- или x➝a-0.

Для обозначения предела справа вместо x➝a пишут x➝a+ или x➝a+0.

Эти пределы характеризуют поведение функции слева и справа от точки a. Их называют односторонними пределами.

Свойства бесконечно больших функций:

1) произведение бескон. большой функции при x➝a на постоянную c≠0 есть бескон. большая функция при x➝a;

2) произведение двух бескон. больших функций при x➝a есть бескон. большая функция при x➝a;

3) произведение бескон. большой функции при x➝a на функцию, имеющую конечый предел в точке a, есть бескон. большая функция при x➝a.

2.8 Эквивалентные функции

Пусть функции f(x) и g(x) определены в некоторой проколотой окрестности точки x₀, пусть g(x)≠0, тогда если предел

1) равен 0, то говорят, что функция f(x) мала по сравнению с g(x) при x➝x₀;

2) равен k≠0, то говорят, что функции f(x) и g(x) - функции одного порядка при x➝x₀;

3) равен 1, то говорят, что функция f(x) эквивалентна функции g(x) при x➝x₀, пишут: f(x)~g(x).

Второе определение непрерывности: пусть функция f(x) определена в некоторой окрестности точки x₀ из области определения функции. Функция f(x) называется непрерывной в точке x₀, если бесконечно малому приращению аргумента соответствует бесконечно малое приращение функции в этой точке.

ТЕОРЕМА об арифметических свойствах непрерывных функций: пусть функции f(x) и g(x) непрерывны в точке x₀, тогда в этой точке непрерывны: сумма f(x)±g(x), произведение f(x)·g(x) и частное f(x)/g(x) этих функций.

ТЕОРЕМА о непрерывности сложной функции: пусть f(y) непрерывна в точке y₀, пусть y=g(x) непрерывна в точке x₀ и g(x₀)=y₀, тогда сложная функция f(g(x)) будет непрерывна в точке x₀.
