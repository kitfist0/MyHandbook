2. Матрицы, основные определения, теорема Лапласа

2.1 Основные определения

Матрицей A размерности m×n называется прямоугольная таблица m×n чисел aij, где i = 1, ... m и j = 1, ... n:

A =
a11  a12  …  a1n
a21  a22  …  a2n
…  …  …  …
am1  am2  …  amn

расположенных в m строках и n столбцах. Матрица называется квадратной, если m = n.

Элементы a₁₁, a₂₂, ... amn образуют главную диагональ матрицы (показана красным на рис. выше), а элементы am1, a(m−1)2, ... a1n – побочную диагональ.

Квадратная матрица, у которой все элементы, кроме элементов главной диагонали равны нулю называется диагональной. Диагональная матрица, у которой каждый элемент главной диагонали равен 1, называется единичной. Квадратная матрица называется треугольной, если все элементы, расположенные по одну сторону от главной диагонали равны нулю. Матрица, все элементы которой равны нулю, называется нулевой.

Матрицы равны между собой, если равны все соответствующие элементы этих матриц.

Матрица, полученная из данной заменой каждой ее строки столбцом с тем же номером (проще говоря: соответствующие строки меняются с соответствующими столбцами местами), называется матрицей транспонированной к данной. Обозначается Aᵀ. Рассмотрим ПРИМЕР транспонирования матрицы:

A =
1 2 3
4 5 6
7 8 9
 ,  Aᵀ =
1 4 7
2 5 8
3 6 9

Для операции транспонирования верны свойства:

1) (A+B)ᵀ = Aᵀ + Bᵀ

2) (AB)ᵀ = Bᵀ ⋅ Aᵀ

2.2 Действия с матрицами

1) Умножение матрицы на число. Чтобы умножить матрицу A на число k, нужно умножить на это число все элементы матрицы A:

 
k ·
 
a11 a12 a13
a21 a22 a23
a31 a32 a33
 
 = 
 
ka11 ka12 ka13
ka21 ka22 ka23
ka31 ka32 ka33

Матрица −A = (−1) ⋅ A называется противоположной матрице А.

2) Суммой двух матриц одинаковой размерности, называется матрица той же размерности, каждый элемент которой равен сумме соответствующих элементов слагаемых.

A =
a11 a12 a13
a21 a22 a23
a31 a32 a33
 ,  B =
b11 b12 b13
b21 b22 b23
b31 b32 b33
A + B =
a11 + b11 a12 + b12 a13 + b13
a21 + b21 a22 + b22 a23 + b23
a31 + b31 a32 + b32 a33 + b33

3) Разность матриц А и В можно определить так:

A − B = A + (−B)

4) Свойства операций сложения матриц и умножения на число:

A + B = B + A

A + (B + C) = (A + B) + C

A + 0 = A, A − A = 0, 1 ⋅ A = A

k ⋅ (A + B) = k ⋅ A + k ⋅ B

5) Произведение матриц определяется следующим образом. Пусть заданы две матрицы A (размерности m×n) и B (размерности n×k), причем число столбцов первой из них равно числу строк второй:

Am×n =
a11  a12  …  a1n
a21  a22  …  a2n
…  …  …  …
am1  am2  …  amn
,  Bn×k =
b11  b12  …  b1k
b21  b22  …  b2k
…  …  …  …
bn1  bn2  …  bnk

то произведением матриц A и B, называется матрица C (размерности m×k):

Cm×k =
c11  c12  …  c1k
c21  c22  …  c2k
…  …  …  …
cm1  cm2  …  cmk

элементы которой вычисляются по формуле:

cij = ai1b1j + ai2b2j + ... + ainbnj

где i=1, ..., m и j=1, ..., k. Произведение матриц A и B обозначается AB, т.е. C = AB.

Иначе говоря, элементы матрицы-произведения определяются следующим образом: элемент i-й строки и j-го столбца матрицы С равен сумме произведений элементов i-й строки матрицы А на соответствующие элементы j-го столбца матрицы В:

ai1 ai2 ... aim
... ... ... ...
● ● ... ●
  · 
● b1j ● ●
● b2j ● ●
... ... ... ...
● bnj ● ●

Имеем: матрица А размера 2×3, матрица В размера 3×3, тогда произведение АВ = С существует и элементы матрицы С равны:

с₁₁ = 1 ⋅ 1 + 2 ⋅ 2 + 1 ⋅ 3 = 8, с₁₂ = 1 ⋅ 2 + 2 ⋅ 0 + 1 ⋅ 5 = 7, с₁₃ = 1 ⋅ 3 + 2 ⋅ 1 + 1 ⋅ 4 = 9,

с₂₁ = 3 ⋅ 1 + 1 ⋅ 2 + 0 ⋅ 3 = 5, с₂₂ = 3 ⋅ 2 + 1 ⋅ 0 + 0 ⋅ 5 = 6, с₂₃ = 3 ⋅ 3 + 1 ⋅ 1 + 0 ⋅ 4 = 10.

Произведение BA не существует, т.к. число столбцов матрицы B не равно числу строк матрицы А.

Произведение матриц в общем случае антикоммутативно, т.е. AB≠BA. Матрицы A и B называются перестановочными, если AB=BA.

6) Умножение матриц обладает следующими свойствами:

A ⋅ (B ⋅ C) = (A ⋅ B) ⋅ C

A ⋅ (B + C) = AB + AC

k ⋅ (A ⋅ B) = (k ⋅ A) ⋅ B

2.3 Элементарные преобразования матриц

Элементарными преобразованиями матриц являются:

1) перестановка местами любых двух строк (столбцов) матрицы;

2) умножение любой строки (столбца) матрицы на константу, отличную от нуля;

3) прибавление к любой строке (столбцу) матрицы другой строки (столбца), умноженной на константу, отличную от нуля.

Две матрицы А и В называются эквивалентными, если одна из них получается из другой с помощью элементарных преобразований. Пишут A~B.

Элементарные преобразования матрицы — это такие преобразования матрицы, в результате которых сохраняется эквивалентность матриц. Следовательно, элементарные преобразования не изменяют множество решений системы линейных алгебраических уравнений, которую представляет эта матрица.

2.3.1 Приведение матриц к ступенчатому виду

В линейной алгебре матрица считается матрицей ступенчатого вида (ступенчатого вида по строкам) если:

1) все ненулевые строки (имеющие по крайней мере один ненулевой элемент) располагаются над всеми полностью нулевыми строками;

2) ведущий элемент (первый ненулевой элемент строки при отсчёте слева направо) каждой ненулевой строки располагается строго правее ведущего элемента в строке, расположенной выше данной.

Всё становится понятнее, если это записать:

1 4 7
0 2 1
0 0 3
c₁
c₂
c₃

Матрица называется матрицей приведенного ступенчатого вида по строкам (или канонического вида по строкам) если она удовлетворяет дополнительному условию: каждый ведущий элемент ненулевой строки - это единица, и он является единственным ненулевым элементом в своём столбце. Вот пример матрицы приведенного ступенчатого вида по строкам:

1 0 0
0 1 0
0 0 1
c₁
c₂
c₃

Важно отметить, что левый край матрицы приведенного ступенчатого вида по строкам не обязательно имеет вид единичной матрицы. Например, следующая матрица является матрицей приведенного ступенчатого вида:

1 0 4 0
0 1 2 0
0 0 0 1
c₁
c₂
c₃

ТЕОРЕМА (о приведении матриц к ступенчатому виду): любую матрицу путём элементарных преобразований только над строками можно привести к ступенчатому виду.

2.4 Определитель матрицы

Определитель (или детерминант) — одно из основных понятий линейной алгебры. Это многочлен, комбинирующий элементы КВАДРАТНОЙ матрицы таким образом, что его значение сохраняется при транспонировании и линейных комбинациях строк или столбцов. Т.е., определитель характеризует содержание матрицы. В частности, если в матрице есть линейно-зависимые строки или столбцы определитель равен нулю. Определитель играет ключевую роль в решении в общем виде систем линейных уравнений, на его основе вводятся базовые понятия.

3) Базисный минор. В матрице порядка m×n минор порядка k называется базисным, если он не равен нулю, а все миноры порядка k+1 и выше равны нулю, или не существуют вовсе, т.е. k совпадает с меньшим из чисел m или n. Столбцы и строки матрицы, на которых стоит базисный минор, также называются базисными. В матрице может быть несколько различных базисных миноров, имеющих одинаковый порядок.

2.4.3 Ранг матрицы

Ранг матрицы — наивысший из порядков миноров этой матрицы, отличных от нуля. Рангом системы строк (столбцов) матрицы A с m строк и n столбцов называется максимальное число линейно независимых строк (столбцов). Несколько строк (столбцов) называются линейно независимыми, если ни одна из них не выражается линейно через другие. Ранг системы строк всегда равен рангу системы столбцов, и это число называется рангом матрицы.

Обычно ранг матрицы A обозначается rang A, rg A или rank A.

ТЕОРЕМА (о базисном миноре): Пусть r = rang A, Mr — базисный минор матрицы A, тогда:

1) базисные строки и базисные столбцы линейно независимы;

2) любая строка (столбец) матрицы A есть линейная комбинация базисных строк (столбцов).

ТЕОРЕМА (об инвариантности ранга при элементарных преобразованиях): если матрицы A и B могут быть получены друг из друга элементарными преобразованиями, то их ранги равны.

Следствия:

1) Если ранг матрицы равен r, то любые p строк или столбцов (p > r) этой матрицы будут линейно зависимы

2) Если A — квадратная матрица, и det(A) = 0, то строки и столбцы этой матрицы линейно зависимы

3) Пусть r = rang A, тогда максимальное количество линейно независимых строк (столбцов) этой матрицы равно r.

МЕТОДЫ нахождения ранга матрицы

1) Метод элементарных преобразований

Ранг матрицы равен числу ненулевых строк в матрице после приведения её к ступенчатой форме при помощи элементарных преобразований над строками матрицы.

2) Метод окаймляющих миноров

Пусть в матрице A найден ненулевой минор k-го порядка M. Рассмотрим все миноры (k+1)-го порядка, включающие в себя (окаймляющие) минор M; если все они равны нулю, то ранг матрицы равен k. В противном случае среди окаймляющих миноров найдется ненулевой, и вся процедура повторяется.

2.4.5 Теорема Лапласа

Теорема Лапласа — одна из теорем линейной алгебры. Названа в честь французского математика Пьера-Симона Лапласа, которому приписывают формулирование этой теоремы в 1772 году, хотя частный случай этой теоремы о разложении определителя по строке (столбцу) был известен ещё Лейбницу (и рассмотрен нами выше).

Теорема Лапласа обобщает формулу разложения определителя по элементам строки (столбца).

Рассмотрим матрицу A размерности n×n

A =
a11  a12  …  a1n
a21  a22  …  a2n
…  …  …  …
an1  an2  …  ann

2.4.6 Теорема Бине-Коши

Формула Бине-Коши — теорема об определителе произведения двух прямоугольных матриц, при условии, что оно является квадратной матрицей.

Произведение двух прямоугольных матриц A и B дает квадратную матрицу порядка m, если A имеет n столбцов и m строк, а матрица B имеет m столбцов и n строк. Миноры матриц A и B одинакового порядка, равного наименьшему из чисел n и m, называются соответствующими друг другу, если они стоят в столбцах (матрицы A) и строках (матрицы B) с одинаковыми номерами.

Am×n · Bn×m = ABm×m

3) равен произведению определителей матриц A и B, если n = m.

2.5 Свойства определителей

1) Определитель, имеющий строку, представленной суммой двух строк, равносилен сумме определителей

   a11     a12
a21 + b21  a22 + b22
 = 
a11   a12
a21   a22
 + 
a11   a12
b21   b22

2) Определитель, имеющий нулевую строку, равен нулю.

3) Определитель, имеющий две пропорцинальных строки, равен нулю.

4) Определитель единичной матрицы равен единице.

5) Опредилетель транспонированной матрицы равен определителю нетранспонированной матрицы: det A = det Aᵀ

6) Определитель не изменится, если из строки вынести множитель.

7) Определитель не изменится, если одну из его строк домножить на число и добавить к другой cтроке.

a11   a12
a21   a22
 = 
    a11      a12
k·a11 + a21  k·a12 + a22

8) При перестановке двух строк определителя, определитель меняет знак.

9) Все свойства сформулированые для строк, верны и для столбцов.

2.6 След матрицы

След определяется для произвольной квадратной матрицы A, как сумма элементов её главной диагонали. В математике встречается два обозначения операции вычисления следа: tr(A) (от англ. trace — след), и Sp(A) (от нем. Spur — след).

Свойства

1) Линейность tr (c₁A + c₂B) = c₁ ⋅ tr A + c₂ ⋅ tr B

2) tr AB = tr BA

3) tr Aᵀ = tr A

4) Определитель квадратной матрицы n×n можно выразить через следы степеней этой матрицы, не превосходящие n. Например:

det A3×3 = (1/6) ⋅ ( (tr A)³ − 3 ⋅ tr A ⋅ tr A² + 2 ⋅ tr A³ )
