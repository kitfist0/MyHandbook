16. Операционное исчисление

Операционное исчисление — один из методов математического анализа, позволяющий в ряде случаев с помощью весьма простых средств решать сложные математические задачи.

Преобразование Лапласа ( ℒ ) — интегральное преобразование, связывающее функцию F(p) комплексного переменного (изображение) с функцией f(t) вещественного переменного (оригинал). С его помощью исследуются свойства динамических систем и решаются дифференциальные и интегральные уравнения.

Одной из особенностей преобразования Лапласа, которые предопределили его широкое распространение в научных и инженерных расчётах, является то, что многим соотношениям и операциям над оригиналами соответствуют более простые соотношения над их изображениями. Так, свёртка двух функций сводится в пространстве изображений к операции умножения, а линейные дифференциальные уравнения становятся алгебраическими.

16.1 Оригинал и изображение

Функцией-оригиналом будем называть любую функцию действительного аргумента t : f (t) = u (t) + i · v (t), удовлетворяющую следующим условиям:

1) f (t), f ' (t) или всюду непрерывны, или имеют на любом конечном промежутке лишь конечное число точек разрыва первого рода;

2) f (t) = 0 для всех точек t<0;

3) f (t) возрастает не быстрее показательной функции, т.е. существуют такие числа M>0 и s≥0, что для всех t выполняется:

| f (t) | ≤ M · e st

где s – показатель роста функции f(t).

Изображением функции f(t) (по Лапласу) называют функцию комплексной переменной F(p) (где p - комплексная переменная), определяемую соотношением

F (p) = ℒ{ f (t) } = o ∫ ∞ f (t) · e −pt dt

Интеграл в формуле выше называют интегралом Лапласа для функции f(t), переход от оригинала f(t) к изображению F(p) – преобразованием Лапласа и обозначают буквой ℒ , обратное преобразование, соответственно, обозначают ℒ⁻¹ .

16.2 Свойства преобразования Лапласа

1) Линейность. Оригинал линейной комбинации функций равен линейной комбинации изображений с теми же коэффициентами.

ℒ { a · f(t) + b · g(t) } = a · F(p) + b · G(p)

где a и b – произвольные комплексные числа.

2) Теорема подобия

ℒ{ f (at) } = (1 / a) · F (p / a), где a>0

3) Дифференцирование оригинала

ℒ { f (t) } = F (p)

ℒ { f ' (t) } = p · F (p) − f (0)

ℒ { f '' (t) } = p² · F (p) − p · f (0) − f ' (0)

ℒ { f ⁽ⁿ⁾ (t) } = pⁿ · F (p) − pⁿ⁻¹ · f (0) − pⁿ⁻² · f ' (0) − ... − f ⁽ⁿ⁻¹⁾ (0)

Именно это свойство и обеспечило такую популярность преобразованию Лапласа: оно операцию дифференцирования оригинала заменяет операцией умножения изображения на p. Это сильно упрощает решение задач, где есть производные.

4) Дифференцирование изображения

ℒ⁻¹ { F ' (p) } = −x · f (t)

ℒ⁻¹ { F ⁽ⁿ⁾ (p) } = (−1)ⁿ · xⁿ · f (t)

5) Интегрирование оригинала

ℒ { o ∫ t f (t) dt } = (1 / p) · F (p)

6) Интегрирование изображения

ℒ⁻¹ { p ∫ ∞ f (z) dz } = f (t) / t

7) Теорема смещения

ℒ { e at · f (t) } = F (p−a)

8) Теорема запаздывания

ℒ { f (t−τ) } = e −pτ · F (p)

9) Теорема умножения (свёртки)

ℒ { o ∫ τ f (τ) · g (t−τ) dt } = F (p) · G (p)

Выражение в левой части называется сверткой функций f(t) и g(t) и обозначают f(t)*g(t). Эта операция часто встречается при решении прикладных задач, и преобразование Лапласа позволяет заменить операцию свертки двух оригиналов операцией умножения их изображений.
