17. Гамма-функция

Гамма-функция - одна из самых важных неэлементарных функций анализа. Это математическая функция, которая расширяет понятие факториала на поле комплексных чисел. Обычно обозначается Г(z). Является мероморфной на комплексной плоскости и имеет полюса в точках: z=0,-1,-2,-3,... (Функция комплексной переменной называется мероморфной, если она определена на всей комплексной плоскости и не имеет в конечной части плоскости особых точек, отличных от полюсов).

Гамма функция находит очень широкое применение в прикладном анализе. В статистике широко используется гамма-распределение, частными случаями которого являются экспоненциальное распределение и распределение хи-квадрат. В данной главе вводится понятие гамма-функции, приведены ее основные свойства, а также показан алгоритм ее численного расчета.


17.1 Определение гамма-функции

В математике вводится понятие факториала для натурального числа: n! = 1·2·3 ... n. При этом можно заметить, что (n+1)! = (n+1)⋅n!. Гамма-функция Г(z) распространяет понятие факториала на дробные, отрицательные и даже комплексные значения аргумента z. Гамма-функция не выражается через элементарные функции. Но, если вещественная часть комплексного числа z положительна, гамма-функция может быть определена как интеграл вида:

Г (z) = o ∫ ∞ f z−1 · e −t dt, z ∈ ℂ: Re(z) > 0

Это определение называется интегральным определением гамма-функции. Следует подчеркнуть, что этот несобственный интеграл сходится абсолютно только при положительных значениях вещественной части комплексного числа z.

На всю комплексную плоскость гамма-функция аналитически продолжается через тождество:

Г (z + 1) = z ⋅ Г (z)

Это равенство можно доказать, применяя интегрирование по частям к интегральному определению гамма-функции, тождество верно для любого комплексного значения z. Данное рекуррентное соотношение является очень важным и используется при расчете гамма-функции.

Исходя из тождества выше и, зная что Г(1) = 1, для всех натуральных значений аргумента n гамма-функции, может быть получено выражение:

Г (n + 1) = n ⋅ Г (n) = ... = n! ⋅ Г (1) = n!

Последующие выражения служат альтернативными определениями гамма-функции.

Определение по Гауссу. Оно верно для всех комплексных z, за исключением 0 и отрицательных целых чисел.

Определение по Эйлеру

Верно для всех комплексных z, за исключением 0 и отрицательных целых чисел.

17.2 Свойства гамма-функции и ее вычисление

Здесь следует привести, формулу дополнения Эйлера, которая широко используется при расчете значений Гамма-функции:

Г (1 − z) ⋅ Г (z) = π / sin (π⋅z)

Из неё вытекает формула умножения Гаусса:

Г (z) ⋅ Г (z + 1 / n) ⋅ ... Г (z + (n − 1) / n) =

= n (1 − 2nz) / 2 ⋅ 2π (n − 1) / 2 ⋅ Г (nz)

которую при n=2 называют формулой удвоения Лежандра:

Г (z) ⋅ Г (z + 1 / 2) = 2 1 − 2z ⋅ (√π) ⋅ Г (2z)

Из формулы дополнения Эйлера можно получить выражение для расчета значений гамма-функции от отрицательного аргумента:

Г (z) = π / (sin (πz) ⋅ Г (1 − z))

Необходимо отметить, что при целых z меньших или равных нулю, синус в знаменателе этой формулы обращается в ноль и гамма-функция претерпевает разрыв. График гамма-функции для вещественного аргумента представлен на рисунке ниже.

Рассмотрим некоторые значения гамма-функции. Зная, что для натурального аргумента n гамма-функции Г(n) = (n−1)!, можно рассчитать ее значения: Г(1) = 0! = 1, Г(2) = 1! = 1, Г(3) = 2! = 2.

Найдем значение гамма-функции при z = 0,5, для этого воспользуемся выражением:

Г(1−0,5) ⋅ Г(0,5) = π / sin(π/2), тогда Г(0,5) = √π.

Найдем Г(1,5), для этого воспользуемся выражением:

Г(1,5) = Г(1 + 0,5) = 0,5 ⋅ Г(0,5) = (√π) / 2.

Найдем Г(−0,5), для этого воспользуемся выражением:

Г(−0,5) = Г(1 − 1,5) = π / (sin(3π/2)⋅Г(1,5)) = 2√π.

Теперь рассмотрим очень важный вопрос, касающийся расчета гамма-функции. Для этого рассмотрим несколько возможных интервалов аргумента z.

Пусть 0 < z < 1, тогда в соответствии со свойствами гамма-функции можно вычислить:

Г (z) = Г (z + 1) / z

где 1 < z + 1 < 2, другими словами значение гамма функции при 0 < z < 1 может быть вычилено через значения гамма функции при 1 < z < 2.

Пусть z > 2, тогда для вычисления значения гамма-функции можно записать:

Г (z) = (z − 1) ⋅ Г (z − 1)

При этом Г (z − 1) = (z − 2) ⋅ Г (z − 2), и если продолжать, то можно свести к вычислению гамма-функции в интервале 1 < z < 2.

Рассмотрим на примере:

Г(4,6) = Г(1+3,6) = 3,6 ⋅ Г(3,6) = 3,6 ⋅ Г(1 + 2,6) = ...

... = 3,6 ⋅ 2,6 ⋅ Г(2,6) = 3,6 ⋅ 2,6 ⋅ Г(1 + 1,6) = 3,6 ⋅ 2,6 ⋅ 1,6 ⋅ Г(1,6)

То есть опять задача свелась к вычислению гамма-функции в интервале 1 < z < 2.

Пусть теперь z < 0, тогда при вычислении Г(1−z) по формуле:

Г (z) = π / (sin (πz) ⋅ Г (1 − z))

можно аналогично рекуррентно вычислять путем сведения к гамма-функции в интервале 1 < z < 2.

Теперь для вычисления гамма-функции необходимо получить алгоритм ее расчета при 1 < z < 2. На практике для этого производят аппроксимацию гамма функции на данном интервале в виде:

Г (z) ≈ 1 + P08 (x) / P18 (x),

где x = z − 1, 1 ⩽ z ⩽ 2, P08(x), P18(x) – полиномы 8-ой степени:

P08 (x) = a₁ ⋅ x + a₂ ⋅ x² + ... + a₈ ⋅ x⁸

P18 (x) = b₁ + b₂ ⋅ x + ... + b₈ ⋅ x⁷ + x⁸

Коэффициенты полиномов аппроксимации подобраны так, чтобы обеспечивать наименьшую ошибку аппроксимации. Значения коэффициентов полиномов приведены ниже:

a1 = 6,65e+4, a2 = −3,61e+4, a3 = −3,14e+4, a4 = 866,97, a5 = 629,33, a6 = −379,8, a7 = 24,77, a8 = −1,716

b1 = −1,15e+5, b2 = −1,35e+5, b3 = 4,76e+3, b4 = 2,25e+4, b5 = −3107,8, b6 = −1015,2, b7 = 315,35, b8 = −30,84

Таким образом, используя полиномиальную аппроксимацию и рекуррентные соотношения можно вычислить значения гамма-функции для любого вещественного аргумента.

В заключение, приведем наиболее известные значения гамма-функции от нецелого аргумента и формулы, помогающие при вычислении соответствующих значений:

Г (1/2) = √π

Г (3/2) = (√π) / 2

Г (1/2 + n) = (√π) ⋅ (2n)! / (4ⁿ ⋅ n!) = (√π) ⋅ (2n − 1)!! / 2ⁿ

Г (1/2 − n) = (√π) ⋅ (−4)ⁿ ⋅ n! / (2n)! = (√π) ⋅ (−2)ⁿ / (2n − 1)!!

Отметим также, что гамма-функция дифференцируема бесконечное число раз, и Г'(x) = ψ(x) ⋅ Г(x), где ψ(x) часто называют пси-функцией, или дигамма-функцией.

Кроме того гамма-функция связана с такими функциями, как: K-функция, G-функция Барнса, Бета-функция, дзета-функция Римана и др.
