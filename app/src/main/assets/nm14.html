15. Векторное поле


15.1 Векторное поле, векторные линии

Если каждой точке M заданной области пространства D (чаще всего двумерного или трехмерного) поставлен в соответствие вектор ā с началом в этой точке, то говорят, что в этой области задано векторное поле.

Средством визуализации векторных полей служат векторные линии.

Векторной называется линия, в каждой точке которой направление касательной совпадает с направлением векторного поля в данной точке. Например, скорость движения частиц жидкости направлена по касательной к ее траектории, поэтому векторными линиями поля скоростей потока жидкости являются траектории ее частиц.

На рисунке выше построены векторы, заданного в декартовой системе координат, векторного поля. Видно, что векторы в разных точках пространства, выстраиваясь в хвост друг к другу, образуют цепочки, близкие к векторным линиям.

ЗАМЕЧАНИЕ: в тех случаях, когда векторное поле задает напряженность силовых полей, например гравитационного, электрического или магнитного поля, векторные линии называют силовыми линиями.

15.2.2 Свойства дивергенции. Источники и стоки векторного поля

Для характеристики точек векторного поля можно использовать значение дивергенции. Если div ā(M) > 0, то данная точка есть источник поля; если div ā(M) < 0, то —- сток поля; если div ā(M) = 0, то в этой точке нет ни источиков, ни стоков (либо они компенсируют друг друга). На рисунке ниже, для наглядности, приведены некоторые примеры точек:

Рассмотрим свойства дивергенции:

1) Линейность: для любых векторных полей F и G и для всех вещественных чисел a и b

div(aF + bG) = a·div(F) + b·div(G)

2) Если φ — скалярное поле, а F — векторное, тогда:

div(φF) = grad(φ)·F + φ·div(F)

3) Свойство, связывающее векторные поля F и G, заданные в трёхмерном пространстве, с ротором:

div(F×G) = rot(F)·G + F·rot(G)

4) Дивергенция от градиента есть лапласиан:

div(grad φ) = Δφ

5) Дивергенция от ротора векторного поля F:

div(rot F) = 0

15.3.2 Свойства ротора

1) Линейность: для любых векторных полей F и G и для любых постоянных чисел a и b cправедливо:

rot(aF+bG) = a·rot(F) + b·rot(G)

2) Если φ — скалярное поле, а F — векторное, тогда:

rot(φ·F) = grad(φ)×F + φ·rot(F)

3) Дивергенция ротора равна нулю:

div(rot F) = 0

4) Ротор ротора равен градиенту дивергенции минус лапласиан:

rot(rot F) = grad(div F) − ∆F

Циркуляцией векторного поля ā(M) по данному замкнутому контуру L называется криволинейный интеграл второго рода, взятый по L:

CL(ā) = ∮L ā(M)dl = ∮L P(x,y,z)dx + Q(x,y,z)dy + R(x,y,z)dz

Рассмотрим свойства циркуляции:

1) Аддитивность: циркуляция по контуру, ограничивающему несколько смежных поверхностей, равна сумме циркуляций по контурам, ограничивающим каждую поверхность в отдельности:

CL(ā) = ∑ CLi(ā)

CL(ā) = ПS (rot ā)


15.5 Специальные векторные поля

В физике большое значение играют векторные поля, для которых дивергенция, ротор или эти величины вместе, обращаются в ноль. Такие поля называются специальными. К ним относятся потенциальное, соленоидальное и Лапласово поля. Рассмотрим каждое их них.


15.5.1 Потенциальное векторное поле

Векторное поле ā(M) называется потенциальным (безвихревым), если существует такая непрерывно дифференцируемая скалярная функция U(M) такая, что

ā = grad U(M)

Функция U(M) называется в этом случае потенциалом векторного поля ā(M).

Потенциальными полями являются магнитное поле, создаваемое прямолинейным проводником, поле притяжения данной массы к неподвижному центру (гравитационное поле), электрическое поле напряженности точечного заряда и др. Потенциальное поле является наиболее простым среди векторных полей, ибо оно определяется одной скалярной функцией U=U(M) независимо от размерности пространства, в котором задано векторное поле.

Ниже приведены свойства потенциальных векторных полей:

1) Если U(M) − потенциал поля, то потенциалом этого поля будет и любая другая функция вида U(M)+const. Т.е. если векторное поле потенциально, то его потенциал определяется с точностью до постоянного слагаемого.

2) Если векторное поле ā(M) задано в односвязной области Q, то необходимым и достаточным условием его потенциальности является обращение в ноль ротора поля в любой точке M этого поля: rot ā(M) = 0, ∀ M∈Q

Если в какой-либо задаче необходимо установить потенциальность векторного поля, то необходимо показать, что его ротор равен нулю.


15.5.2 Соленоидальное векторное поле

Пусть в некоторой области Q задано векторное поле ā(M), тогда оно называется соленоидальным (трубчатым), если во всех точках области Q дивергенция векторного поля равна нулю.

div ā(M) = 0, ∀ M∈Q

Соленоидальными полями являются магнитное поле, создаваемое прямолинейным проводником, вдоль которого проходит электрический ток, поле линейных скоростей вращающегося твердого тела, поле линейных скоростей стационарного потока несжимаемой жидкости, не имеющее источников и стоков, и др.

Согласно определению, соленоидальные поля не содержат ни источников, ни стоков. Ниже приведены свойства соленоидальных полей:

1) Из формулы Остроградского–Гаусса следует, что если векторное поле ā(M) соленоидальное, то поток вектора счерез любую замкнутую поверхность S равен нулю.

2) Принцип сохранения интенсивности векторной трубки. Потоки соленоидального векторного поля через различные сечения векторной трубки равны между собой.

3) В соленоидальном векторном поле векторные линии не могут ни начинаться, ни оканчиваться внутри поля. Они либо замкнуты, либо начинаются и оканчиваются на границе поля, либо имеют бесконечные ветви (в случае неограниченного поля).

4) В односвязной области в случае соленоидального векторного поля поток вектора ā(M) через любую поверхность S, опирающуюся на замкнутый контур L, зависит не от вида этой поверхности, а только от самого контура L.


15.5.3 Гармоническое (Лапласово) векторное поле

Векторное поле ā(M), M∈Q называется гармоническим (Лапласовым), если оно является как потенциальным, так и соленоидальным, т.е rot ā(M) = 0, div ā(M) = 0

Если вектор ā(M) удовлетворяет первому из условий, то существует скалярная функция U=U(M), такая, что ā = grad U(M) = 0

Тогда второе из условий приводит к уравнению Лапласа: div ā(M) = div grad U(M) = 0 или ∂²U/∂x² + ∂²U/∂y² + ∂²U/∂z² = 0

Таким образом, гармоническое векторное поле описывается скалярной функцией U=U(M), которая является в области Q решением уравнения Лапласа и называется гармонической функцией.

ПРИМЕР: гармоническими функциями являются:

1) U(x,y,z) = Ax + By + Cz (задана в любой конечной области);

2) U(x,y) = x/(x2+y2) (задана в любой области, не содержащей начала координат).


15.6 Теорема разложения Гельмгольца

Теорема разложения Гельмгольца — утверждение о разложении произвольного дифференцируемого векторного поля на две компоненты:

Если дивергенция и ротор векторного поля F(r) определены в каждой точке конечной открытой области V пространства, то всюду в V функция может быть представлена в виде суммы безвихревого поля F1(r) и соленоидального поля F2(r): F(r) = F1(r)+F2(r), где rot F1(r) = 0, div F2(r) = 0 для всех точек r области V.

Формулировка теоремы: пусть F — векторное поле в трёхмерном прострастве R, и пусть оно дважды непрерывно дифференцируемо и убывает быстрее чем 1/r на бесконечности в случае неограниченной области. Тогда поле F представимо в виде суммы безвихревого поля (ротор которого равен нулю) и соленоидального поля (дивергенция которого равна нулю).
