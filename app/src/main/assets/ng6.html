7. Прямая, уравнение прямой


7.1 Общее уравнение прямой

Любая прямая на плоскости в декартовых координатах может быть задана уравнением первого порядка

Ах + Ву + С = 0,

причем постоянные А, В не равны нулю одновременно. Это уравнение первого порядка называют общим уравнением прямой.

Вектор с координатами (A, B) называется нормальным вектором и он перпендикулярен прямой. Вектор с координатами (−B, A) или (B, −A) называется направляющим вектором, он перпендикулярен нормальному вектору. В зависимости от значений постоянных А, В и С возможны следующие частные случаи:

• C = 0, А ≠ 0, В ≠ 0 – прямая проходит через начало координат

• А = 0, В ≠ 0, С ≠ 0 – прямая параллельна оси Ох

• В = 0, А ≠ 0, С ≠ 0 – прямая параллельна оси Оу

• В = С = 0, А ≠ 0 – прямая совпадает с осью Оу

• А = С = 0, В ≠ 0 – прямая совпадает с осью Ох

Уравнение прямой может быть представлено в различном виде в зависимости от каких – либо заданных начальных условий.


7.2 Уравнение прямой с угловым коэффициентом

Если общее уравнение прямой Ах + Ву + С = 0 привести к виду: y = –A/B ⋅ x – C/B

Обозначим k = –A/B и b = –C/B т.о. получим уравнение y = k ⋅ x + b

Полученное уравнение называется уравнением прямой с угловым коэффициентом.

Прямая линия, пересекающая ось Oy в точке (0, b) и образующая угол φ с положительным направлением оси Ox (см. рисунок ниже):

y = k ⋅ x + b, k = tg(φ)

данное уравнение называется уравнением прямой с угловым коэффициентом. Коэффициент k называется угловым коэффициентом прямой и определяется, как тангенс угла φ между прямой и положительным направлением оси Ox.


7.3 Уравнение прямой в отрезках

Если в общем уравнении прямой Ах + Ву + С = 0, С ≠ 0, то, разделив на (–С), получим:

–A/C ⋅ x – B/C ⋅ y = 1

Обозначим: a = –C/A, b = –C/B

Т.о. получим уравнение:

x/a + y/b = 1 (a ≠ 0, b ≠ 0)

Оно описывает прямую линию, пересекающую ось Ox в точке (a, 0) и ось Oy в точке (0, b) (см. рисунок выше) и называется уравнением прямой в отрезках.

В этом виде невозможно представить прямую, проходящую через начало координат.

7.8 Уравнение прямой в полярных координатах

Уравнение прямой в полярных координатах ρ и φ:

Напомним одну аксиому: если две плоскости в пространстве имеют общую точку, то они имеют общую прямую, на которой находятся все общие точки этих плоскостей. Таким образом, прямую линию в пространстве можно задать, указав две плоскости, пересекающиеся по этой прямой.

Пусть в трехмерном пространстве зафиксирована прямоугольная система координат Oxyz и известно, что прямая L является линией пересечения двух плоскостей α и β, которым отвечают общие уравнения плоскости вида A₁x + B₁y + C₁z + D₁ = 0 и A₂x + B₂y + C₂z + D₂ = 0 соответственно. Так как прямая L представляет собой множество всех общих точек плоскостей α и β, то координаты любой точки прямой L будут удовлетворять одновременно и уравнению A₁x + B₁y + C₁z + D₁ = 0 и уравнению A₂x + B₂y + C₂z + D₂ = 0, координаты никаких других точек не будут удовлетворять одновременно обоим уравнениям плоскостей.
