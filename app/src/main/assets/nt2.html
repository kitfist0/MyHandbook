Решение: событие A – студент ответил на три вопроса; событие Ai – студент ответил на i-ый вопрос i = 1, 2, 3.

3.3 Независимые случайные события

События A и B называются независимыми, если (определение независимости двух событий через определение условной вероятности):
P(A/B) = P(A).

Если случайное событие A не зависит от B, то и B не зависит от A.

События A и B называются независимыми, если P(AB) = P(A)·P(B)

Случайные события A1, A2 ... An называются независимыми в совокупности, если для любого набора значений индексов 1 ≤ i < j < k < ... ≤ n выполняется:
P(AiAj) = P(Ai)·P(Aj)
P(AiAjAk) = P(Ai)·P(Aj)·P(Ak)
...
P(AiAj ... An) = P(Ai)·P(Aj) ... P(An)
Из независимости в совокупности следует попарная независимость. Из попарной независимости не следует независимость в совокупности.

ПРИМЕР 3: получено письмо, которое могло быть отправлено из Коврова или из Пскова. На почтовом штемпеле станции отправления можно разобрать только две рядом стоящие буквы "ов". Какова вероятность того, что письмо отправлено из Коврова?
Решение: Событие A – получено письмо. H1 – письмо из Коврова, H2 – письмо из Пскова.
P(H1) = 0,5; P(H2) = 0,5; P(A/H1) = 2/5 = 0,4; P(A/H2) = 1/4 = 0,25.
P(H1/A) = (0,5·0,4)/(0,5·(0,4 + 0,25)) = 8/13

ПРИМЕР 4: бросили игральную кость. Какова вероятность того, что выпало простое число очков, если известно, что число выпавших очков чётное?
Решение: событие A – выпало простое число очков, B – число выпавших очков чётное. Требуется найти условную вероятность P(A/B)=P(AB)/P(B). Общее количество возможных элементарных исходов n = 6. Количество исходов, благоприятствующих событию B, равно mB = 3 (это очки 2, 4 и 6 – чётные). Количество исходов, благоприятствующих событию A/B (выпало простое число очков И число выпавших очков чётное), равно mAB = 1 (это число 2 – чётное и простое). Следовательно:
P(A/B) = (mAB/n)/(mB/n) = 1/3.

3.5 Геометрическая вероятность

Пусть для данного случайного эксперимента пространство элементарных исходов Ω – непрерывное n-мерное множество, элементарные исходы – элементы этого n-мерного множества. Случайное событие A – область в Rn.

Величина, вычисляемая по формуле:
P(A) = mes(A)/mes(Ω)
называется геометрической вероятностью случайного события A. Здесь mes() обозначает меру рассматриваемого пространства: в одномерном пространстве это длина, в двумерном пространстве это площадь, в трёхмерном – объём.
