6. Векторная алгебра


6.1 Основные определения

Геометрический вектор (или просто вектор) – это отрезок АВ, на котором задано направление, например, от А к В, и обозначаемый AB

Точки А и В называются соответственно началом и концом вектора.

Длиной (модулем) вектора называется расстояние между его началом и концом, она обозначается |AB|

Два вектора называются равными, если они одинаково направлены и имеют одинаковые длины.


ЛЕММА о равенстве двух векторов:

Для любых четырех точек пространства A, B, C и D

AB = CD тогда и только тогда, когда AC = BD

Свойства равенства векторов:

• свойство симметричности, если AB = CD, то CD = AB

• свойство транзитивности, если AB = CD, CD = EF, то AB = EF

Нулевым вектором называется вектор, начало и конец которого совпадают: 0 = AA = BB = ...

Для произвольного вектора a = AB, вектор BA = –a называется противоположным.

Два ненулевых вектора называются коллинеарными, если они лежат на параллельных прямых или на одной прямой. Три вектора (или большее их количество) называются компланарными, если они, будучи приведенными к общему началу, лежат в одной плоскости.


6.2 Линейные операции над векторами


6.2.1 Cумма и разность векторов

Правило треугольника, сложения векторов:

от произвольной точки А пространства отложим первый вектор a = AB

от полученной точки В отложим второй вектор b = BC

тогда сумма двух этих векторов определяется: a + b = AC

Это правило называется правилом треугольника сложения векторов и выражается формулой: AB + BC = AC

Ниже приведен поясняющий рисунок:

Разностью векторов a и b называется вектор: a + (–b)

Можно доказать, что c = a – b, b + c = a

Правило параллелограмма сложения и вычитания векторов:

отложим от точки A два вектора: a = AD, b = AB

достроим до параллелограмма: ABCD

тогда: a + b = AC, a - b = BD


6.2.2 Умножение вектора на число

Произведение числа (скаляра) λ ∈ R на вектор a – есть вектор b = λ ⋅ a, длина которого |b| = |λ| ⋅ |a|, а направление определяется так:

• если λ = 0 или a – нулевой вектор, то и результирующий вектор b – нулевой

• если a – ненулевой вектор, то веткор b одинаково направлен с вектором a (символически b↑↑a) при λ > 0, и противоположно направлен (символически b↓↑a) при λ < 0.


6.2.3 Свойства операций сложения векторов и умножения их на числа

1) a + b = b + a

2) (a + b) + c = a + (b + c)

3) a + 0 = a

4) a + (–a) = 0

5) λ ⋅ (a + b) = λ ⋅ a + λ ⋅ b

6) (λ + μ) ⋅ a = λ ⋅ a + μ ⋅ a

7) (λμ) ⋅ a = λ ⋅ (μ ⋅ a)

Благодаря свойствам 1 и 2 можно складывать любое количество векторов в произвольном порядке.

Правило многоугольника сложения нескольких векторов a₁, a₂, ... an:

От произвольной точки A₀ отложим первый вектор a₁ = A₀A₁,

от его конца A₁ отложим второй вектор a₂ = A₁A₂

и т.д. от конца An-1 предпоследнего вектора отложим последний вектор an = An-1An

Тогда: a₁ + a₂ + ... + an = A₀An


6.3 Разложение вектора по базису

Множество векторов на прямой назовем одномерным векторным пространством, множество векторов на плоскости − двумерным векторным пространством, в пространстве − трехмерным векторным пространством. Легко проверить, что если L − некоторое векторное пространство и a, b ∈ L, то a + b ∈ L и λ ⋅ a ∈ L, где λ – некоторое число.

Линейной комбинацией векторов a₁, a₂, ... an с коэффициентами λ₁, λ₂, ... λn называется вектор: λ₁ ⋅ a₁ + λ₂ ⋅ a₂ + ... + λn ⋅ an

Будем говорить, что вектор b раскладывается по векторам a₁, a₂, ... an, если b является линейной комбинацией этих векторов.

Если a ≠ 0 то любой вектор b, коллинеарный a, представим и причем единственным образом в виде b = λ ⋅ a, где λ − некоторое число.

Пусть a и b – два неколлинеарных вектора. Тогда любой вектор c, компланарный с векторами a и b, раскладывается по ним, причем единственным образом.

Пусть a, b и c − некомпланарные векторы. Тогда любой вектор d раскладывается по этим векторам.

Базисом векторного пространства L будем называть упорядоченную систему векторов пространства, состоящую: из одного ненулевого вектора, если пространство одномерное; из двух неколлинеарных векторов, если пространство двумерное; из трех некомпланарных векторов, если пространство трехмерное.

Более распространенное определение базиса выглядит следующим образом:

Базисом векторного пространства L называется такая упорядоченная линейно независимая система векторов, что любой вектор пространства L раскладывается по векторам этой системы. (линейная зависимость и независимость векторов рассмотрена в пункте 6.4)

Очевидно, что в любом векторном пространстве можно выбрать бесконечно много базисов, число векторов в каждом из них равно размерности пространства. Слова "упорядоченная система векторов" означают, что указан порядок перечисления векторов.

Координатами (или компонентами) вектора a в базисе i₁, i₂, ... ik называются коэффициенты разложения λ₁, λ₂ ... λk вектора a по векторам базиса.

a = λ₁ ⋅ i₁ + λ₂ ⋅ i₂ + ... + λk ⋅ ik

Для указания, что вектор a имеет координаты λ₁, λ₂ ... λk, принято записывать: a = (λ₁, λ₂ ... λk)

Очевидно, что в фиксированном базисе каждый вектор имеет свой, единственный, набор координат.

ВАЖНО: сложение векторов и умножение их на число связаны с аналогичными действиями с их координатами.

При умножении вектора на число все его координаты умножаются на это число: c ⋅ a = (c ⋅ λ₁, c ⋅ λ₂ ... c ⋅ λk)

При сложении векторов складываются их соответствующие координаты: a = (λ₁, λ₂ ... λk), b = (β₁, β₂ ... βk)

c = a + b = (λ₁ + β₁, λ₂ + β₂ ... λk + βk)


6.4 Линейная зависимость и независимость векторов

Система векторов a₁, a₂, ... ak называется линейно зависимой, если существует такой набор коэффициентов λ₁, λ₂ ... λk, из которых хотя бы один отличен от нуля, что λ₁ ⋅ a₁ + λ₂ ⋅ a₂ + ... + λk ⋅ ak = 0

Система векторов a₁, a₂, ... ak называется линейно независимой, если равенство λ₁ ⋅ a₁ + λ₂ ⋅ a₂ + ... + λk ⋅ ak = 0 возможно только при λ₁ = λ₂ = ... = λk = 0.

Система векторов a₁, a₂, ... ak линейно зависима тогда и только тогда, когда один из векторов системы является линейной комбинацией остальных векторов этой системы.

Если система векторов содержит линейно зависимую подсистему, то вся система линейно зависима.

• Система, состоящая из одного вектора, линейно зависима тогда и только тогда, когда этот вектор нулевой.

• Система, состоящая из двух векторов, линейно зависима тогда и только тогда, когда эти векторы коллинеарны.

• Система из трех векторов линейно зависима тогда и только тогда, когда эти векторы компланарны.

• Четыре вектора всегда образуют линейно зависимую систему.


6.5 Разложение вектора по ортам координатных осей

Орты (орт-векторы) осей OX, OY, OZ — это единичные попарно ортогональные векторы: i, j, k с общим началом в начале координат; направления ортов совпадают с направлением осей.

Система ортов (или базисная система векторов) - это система единичных векторов осей координат.

Базис, образованный единичными попарно ортогональными векторами, называют ортонормированным.

Разложим вектор a по этому базису.

Таким образом заданный в пространстве вектор: a = (ax, ay, az) раскладывается по ортам следующим образом: a = ax ⋅ i + ay ⋅ j + az ⋅ k

Нахождение координат вектора через координаты его начала и конца

Зная координаты начала и координаты конца вектора, можно определить координаты самого вектора. Если точки заданы своими координатами A(xa, ya, za), B(xb, yb, zb), то AB = (xb - xa, yb - ya, zb - za)

Таким образом, чтобы найти координаты вектора, нужно из координат его конца вычесть координаты его начала.

Длина вектора через его координаты

Длина вектора a = (ax, ay, az) заданного своими координатами, находится по формуле: |a| = (ax² + ay² + az²)½

Длина (модуль) вектора, координаты которого известны, равен корню квадратному из суммы квадратов координат.


6.6 Проекция вектора

Пусть в пространстве задана некоторая ось x. Проекцией точки A на ось x называется число, соответствующее основанию перпендикуляра, опущенного на ось x из точки A. Проекцией вектора a на ось x называется разность проекций конца вектора и его начала.

На рисунке выше проекция вектора показана синим. Проекцию будем обозначать: Прx a

Проекция вектора a на ось x равна bx−ax. Пусть φ - угол, образованный вектором a с осью x. Тогда Прx a = |a| ⋅ cos(φ)

Проекция на ось x суммы нескольких векторов равна сумме их отдельных проекций.

Проекция на ось x вектора, умноженного на число, равна произведению проекции вектора на это число.

Проекция вектора на другой вектор

Проекцией вектора b на вектор a называют проекцию вектора b на любую ось, параллельную вектору a и имеющую направление, совпадающее с направлением вектора a.

Проекция вектора b на вектор a обозначается: Прa b

Очевидно, что Прa b = |b| ⋅ cos(φ), где φ − угол между векторами a и b.

Проекции вектора на координатные оси равны коодинатам вектора.


6.7 Направляющие косинусы

Направляющими косинусами вектора называются косинусы углов, образованных вектором с положительными направлениями осей координат.

В соответствии с рисунком выше направляющими косинусами вектора a являются cos(α), cos(β), cos(γ), где α − угол между осью x и вектором a, β − угол между осью y и вектором a, γ − угол между осью z и вектором a.

Таким образом направление вектора однозначно задается направляющими косинусами. Для единичного вектора направляющие косинусы равны его координатам.

Если в пространстве задан вектор a = (ax, ay, az) то его направляющие косинусы вычисляются по формулам:

cos(α) = ax / |a|, cos(β) = ay / |a|, cos(γ) = az / |a|

Основное свойство направляющих косинусов

Сумма квадратов направляющих косинусов равна единице: cos²α + cos²β + cos²γ = 1.

Если известны направляющие косинусы вектора a = (ax, ay), то его координаты могут быть найдены по формулам:

ax = |a| ⋅ cos(α)

ay = |a| ⋅ cos(β)

Аналогичные формулы имеют место и в трехмерном случае - если известны направляющие косинусы вектора a = (ax, ay, az), то его координаты могут быть найдены по формулам:

ax = |a| ⋅ cos(α)

ay = |a| ⋅ cos(β)

az = |a| ⋅ cos(γ)


6.8 Скалярное произведение векторов

Скалярное произведение (иногда внутреннее произведение) — операция над двумя векторами, результатом которой является число (скаляр), не зависящее от системы координат и характеризующее длины векторов-сомножителей и угол между ними.

Скалярным произведением двух ненулевых векторов a и b называется число, равное произведению длин этих векторов на косинус угла между ними:

a ⋅ b = |a| ⋅ |b| ⋅ cos(φ)

Скалярное произведение векторов обозначают: (a, b)

Если хотя бы один из векторов в скалярном произведении равен нулевому вектору, то скалярное произведение равно нулю. Если угол между двумя векторами острый, то их скалярное произведение положительно; если угол между векторами тупой, то скалярное произведение этих векторов отрицательно. Скалярное произведение двух ненулевых векторов равно нулю, тогда и только тогда, когда эти векторы ортогональны (взаимно перпендикулярны). Ортогональными (перпендикулярными) называются векторы, скалярное произведение которых равно нулю.

Свойства скалярного произведения

1) (a, b) = (b, a)

2) (a, a) = |a|²

3) если a ≠ 0, то (a, b) = |a| ⋅ Прa b

4) если a ≠ 0, b ≠ 0 и (a, b) = 0, то a⊥b

5) (a + b, c) = (a, c) + (b, c)

6) (λ ⋅ a, b) = λ ⋅ (a, b)

7) (α ⋅ a + β ⋅ b, γ ⋅ c + δ ⋅ d) = αγ ⋅ (a, c) + αδ ⋅ (a, d) + βγ ⋅ (b, c) + βδ ⋅ (b, d)

Если векторы a и b заданы своими координатами: (xa, ya, za) и (xb, yb, zb) соответственно, то их скалярное произведение вычисляется по формуле: (a, b) = xaxb + yayb + zazb

Скалярное произведение векторов, заданных своими координатами, равно сумме произведений соответствующих координат.

6.9 Векторное произведение векторов

Векторным произведением вектора a на вектор b называется вектор c, удовлетворяющий следующим требованиям:

1) длина вектора b равна произведению длин векторов a и b на синус угла φ между ними: |c| = |a| ⋅ |b| ⋅ sin(φ)

2) вектор c ортогонален каждому из векторов a и b

3) вектор c направлен так, что тройка векторов a b c является правой

Векторное произведение обозначают: [a, b] или a×b

Тройка векторов a b c называется левой, если кратчайший поворот от вектора a (первого вектора тройки) к вектору b (второму вектору тройки), видимый с конца третьего вектора c, осуществляется по часовой стрелке.

Тройка векторов a b c называется правой, если кратчайший поворот от вектора a (первого вектора тройки) к вектору b (второму вектору тройки), видимый с конца третьего вектора c, осуществляется против часовой стрелки.

Свойства векторного произведения:

1) Векторное произведение двух векторов есть нулевой вектор [a, b] = 0 тогда и только тогда, когда a||b

2) Векторное произведение антикоммутативно, то есть: a×b = –b×a

3) Модуль векторного произведения равен площади параллелограмма, построенного на заданных векторах a и b, т.е.

4) Свойство ассоциативности относительно умножения на скаляр [λa, b] = [a, λb] = λ ⋅ [a, b]

5) Свойство дистрибутивности по сложению

[a₁ + a₂, b] = [a₁, b] + [a₂, b]

[a, b₁ + b₂] = [a, b₁] + [a, b₂]

6.10 Смешанное произведение векторов

Смешанным произведением трех векторов (a, b, c) называется число, равное скалярному произведению вектора a×b на вектор c: (a, b, c) = ([a, b], c)

Свойства смешанного произведения

1) Смешанное произведение кососимметрично по отношению ко всем своим аргументам: (a, b, c) = (b, c, a) = (c, a, b) = –(b, a, c) = –(a, c, b) = –(c, b, a)

т.е. перестановка любых двух сомножителей меняет знак произведения. Отсюда следует, что: ([a, b], c) = (a, [b, c])

Смешанное произведение (a, b, c) в правой декартовой системе координат (в ортонормированном базисе) равно определителю матрицы, составленной из координат векторов a, b и c:

3) Три вектора компланарны тогда и только тогда, когда их смешанное произведение равно нулю

4) Тройка векторов a, b, c является правой тогда и только тогда, когда их смешанное произведение больше нуля. Если же оно меньше нуля, то векторы a, b и c образуют левую тройку.

5) Свойство ассоциативности относительно умножения на скаляр

(λa, b, c) = (a, λb, c) = (a, b, λc) = λ ⋅ (a, b, c)

6) Свойство дистрибутивности по сложению

(a₁ + a₂, b, c) = (a₁, b, c) + (a₂, b, c)

(a, b₁ + b₂, c) = (a, b₁, c) + (a, b₂, c)

(a, b, c₁ + c₂) = (a, b, c₁) + (a, b, c₂)

Геометрический смысл смешанного произведения

Смешанное произведение (a, b, c) по абсолютному значению равно объёму параллелепипеда (смотри рисунок), образованного векторами a, b и c

знак зависит от того, является ли эта тройка векторов правой или левой. Объем тетраэдра, построенного на векторах a, b и c, равен одной шестой объема соответствующего параллелепипеда.
